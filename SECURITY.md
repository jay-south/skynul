# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use GitHub Security Advisories for private reporting:

- In the repository, go to Security -> Advisories -> "Report a vulnerability".

If that is not available, open an issue with minimal details and ask for a
private channel.
