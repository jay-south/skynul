# Contributing

Thanks for taking the time to contribute.

## Development

Install dependencies:

```bash
pnpm install
```

Run the dev app:

```bash
pnpm dev
```

Quality checks:

```bash
pnpm lint
pnpm typecheck
```

## Pull Requests

- Keep PRs focused and small.
- Explain the "why" and include reproduction steps for bug fixes.
- Do not include secrets or credentials.
